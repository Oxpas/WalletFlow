
# 📱 WalletFlow

WalletFlow — нативное iOS приложение для отслеживания подписок и регулярных платежей, которое помогает пользователям экономить деньги и контролировать расходы.

Приложение позволяет добавлять любые подписки и счета, получать уведомления до списаний, анализировать расходы и видеть, куда уходят деньги каждый месяц. Данные хранятся локально и могут синхронизироваться через iCloud.

## 🚀 Возможности

- 📊 **Трекер подписок и платежей** — добавляй регулярные расходы в одном месте.
- ⏰ **Напоминания** — получай уведомления за несколько дней до списания.
- 📅 **Календарь платежей** — визуальный график всех предстоящих затрат.
- 📈 **Аналитика расходов** — красивые графики расхода по категориям и периодам.
- 🎨 **Нейроморфный дизайн** — современный UI, поддержка тёмной и светлой тем.

## ⚙️ Установка

1. Клонируй репозиторий:
   ```bash
   git clone git@github.com:Oxpas/WalletFlow.git
   ```
2. Открой проект в Xcode:
   ```bash
   open WalletFlow.xcodeproj
   ```
3. Выбери целевое устройство или симулятор и запусти приложение (Cmd + R).
